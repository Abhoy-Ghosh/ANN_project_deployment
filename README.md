# ANN Project Deployment - Customer Churn Prediction

A Deep Learning based Customer Churn Prediction system built using TensorFlow, Scikit-Learn, and Streamlit. The application predicts the probability of a bank customer leaving the service based on demographic and account-related features.

## Features

* Customer churn prediction using ANN
* Real-time predictions through Streamlit UI
* Data preprocessing with Label Encoding, One-Hot Encoding, and Standard Scaling
* Deployed on Streamlit Cloud

## Tech Stack

* Python
* TensorFlow / Keras
* Scikit-Learn
* Pandas & NumPy
* Streamlit

## Live Demo
